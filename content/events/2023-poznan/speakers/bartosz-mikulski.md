#### Speaker

Bartosz Mikulski is an MLOps engineer, blogger, and conference speaker.

Bartosz has been blogging about machine learning and data engineering since 2017 (https://mikulskibartosz.name). He contributed one chapter to the book "97 Things Every Data Engineer Should Know."
