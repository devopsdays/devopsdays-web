Cluster API é um subprojeto Kubernetes que permite provisionar e gerenciar simultaneamente clusters K8s em multiplos provedores de infraestrutura. Justamente com o Cluster Autoscaler formam uma incrível combinação de infraestrutura como código (IaC) e automação através de APIs declarativas.
