Cet atelier est une session intensive et pratique exclusivement axée sur Docker. Il est conçu pour une gamme de participants, autant à ceux qui sont nouveaux à Docker qu'à ceux qui cherchent à améliorer leurs compétences existantes. L'atelier couvrira les fondamentaux de Docker, les meilleures pratiques, les fonctionnalités avancées et des cas d'usage pratiques, en faisant une session complète pour maîtriser Docker dans le domaine de la conteneurisation.
