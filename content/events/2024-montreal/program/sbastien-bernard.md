Dans cet atelier, nous allons analyser la dette technique d’un projet Java Spring Boot et utiliser Open Rewrite ainsi que GitHub Copilot afin de résoudre les problèmes.

À la fin de cet atelier, vous aurez acquis les compétences nécessaires pour moderniser vos projets à la vitesse supérieure.

Connaissance nécessaire :
Java, Spring Boot

Outils qui seront explorés :
GitHub Copilot, Open Rewrite, SonarQube
