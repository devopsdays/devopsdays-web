Internal developer portals have been a recent trend in the platform engineering space that many developers have not had the chance to experience.

This workshop will involve setting up your own Backstage internal developer portal and exploring some key functionality:
- Software Catalogs
- Tech Docs
- Software Templates
- Plugins

All you need is a laptop with Visual Studio Code, Docker, and access to Github!

Prerequisites:
- A laptop
- Up to date Visual Studio Code
- Up to date Docker
- Access to Github
