No projeto que já estou a 3 anos, tinhamos o desafio de servir as soluções da empresa para usuários Chineses com o mínimo de latência e o máximo de disponibilidade, para isto, não podíamos utilizar Cloud comum, nem regions em Honk Kong, devido ao Firewall Chinês, que é um firewall de entrada e saída que todos os dados que vão para Mainland (território de administração do governo chinês) passam. Utilizamos AWS China para servir os usuários, falaremos da diferença dela para a AWS normal, e como hoje temos um ambiente multicloud com GCP para usuários globais e AWS para usuários na China.
