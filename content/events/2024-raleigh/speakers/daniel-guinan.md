Daniel is a member of the System Test team responsible for testing WebSphere Application Server at IBM.  In his 3 years at IBM he has automated testing using GitOps, as well as containerized legacy applications to be run on Kubernetes. He also manages various long running test applications on production-like on-prem application server environments as well as on OpenShift clusters.  In his free time he enjoys exercising as well as working on his home lab.
