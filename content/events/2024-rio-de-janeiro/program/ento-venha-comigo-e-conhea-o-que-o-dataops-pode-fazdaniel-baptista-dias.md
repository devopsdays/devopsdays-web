Nesta palestra quero explicar ao público alguns conceitos de Observabilidade e apresentar como o OpenTelemetry nos ajuda com ferramentas para gerar métricas e traces de observabilidade para monitorar e depurar sua aplicação em ambientes de produção. Pretendo mostrar como o OpenTelemetry SDK funciona para Go e como enviar métricas e rastreamentos para avaliar sua API e testá-la usando testes baseados em rastreamento para avaliar cenários técnicos e de negócios.Sou embaixador da CNCF e colaborador do projeto OpenTelemetry, especificamente no OpenTelemetry Demo (https://github.com/open-telemetry/opentelemetry-demo), e escrevi dois artigos no blog OpenTelemetry mostrando como usar o OpenTelemetry para monitorar Kubernetes (https: //opentelemetry.io/blog/2023/k8s-runtime-observability/) e sobre a execução de testes baseados em Trace no OpenTelemetry Demo (https://opentelemetry.io/blog/2023/testing-otel-demo/).Atualmente sou desenvolvedor na Kubeshop, mantendo o Tracetest (https://github.com/kubeshop/tracetest), uma ferramenta de código aberto escrita em Go que permite aos usuários usar o OpenTelemetry Traces para construir e executar testes de API e2e de forma simples. e maneira mais rápida.
