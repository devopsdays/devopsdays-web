Infraestrutura como código já é uma realidade no dia-a-dia de quem trabalha com Plataforma.

Mas muitas empresas encontram desafios de como escalar esse tipo de tecnologia para que os times de desenvolvimento possam utilizar de forma autônoma.

É isso que o Backstage pode trazer para nós através do Software Template, para os plataforma é um YAML para os times de desenvolvimento é somente um formulário.
