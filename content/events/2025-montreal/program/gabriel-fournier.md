Git is a super tool in any developer's toolkit.

One thing where git is less than perfect is commit authentication.

Enter Git Commit Signing, to cryptographically authenticate a commit.
And more importantly protect yourself from someone else sneaking a malicious commit under your name.

In this talk, we'll discuss what is Git Commit Signing, why and how you should implement them in your code bases.
We'll also discuss how these commit signatures saved me from angry lawyers after one mishap downed production after my termination.
