### Workshop

🔥 Capture The Flag (CTF): Hack, Secure, Compete! 🔥
Join us for an action-packed Capture The Flag (CTF) event, where developers and security enthusiasts compete in live hacking challenges. This hands-on competition will push your security skills to the limit as you navigate real-world security scenarios, uncover vulnerabilities, and learn how to secure modern applications.

#### Target Audience

🛡 Who Should Participate?
*  Developers, DevOps Engineers, and Security Enthusiasts
*  Ethical Hackers and Bug Bounty Hunters
*  Anyone interested in improving their security skills!

#### Workshop Agenda (90 min)

*  ⁠CTF Rundown and demo (15 min)
*  ⁠CTF begins compete for prizes (70 min)
*  Wrap up and winners announced (5 min)

#### Workshop Overview

🎯 What to Expect
1. **Live Hacking Challenges** – Exploit security flaws in web applications, cloud misconfigurations, open-source dependencies, and more.
2. **Real-World Security Scenarios** – Gain hands-on experience in Remote Code Execution (RCE), privilege escalation, API security, and container security.
3. **Compete & Learn** – Earn points by identifying vulnerabilities, then learn how to fix them using Snyk’s powerful security tools.
4. **Expert Insights & Live Demos** – Get guidance from security experts, watch real-time exploit demonstrations, and explore best practices in DevSecOps.
5. **Exclusive Prizes & Recognition** – The top performers will win awesome prizes and gain recognition in the security community!
